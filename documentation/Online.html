# Online calculation platform

# Biophotonics Diagnostics (BPD) customers

BPD users can request trial access to an online RAMANMETRIX version at the Biophotonics Diagnostics GmbH homepage bpd.gmbh/en/demo-licensesopen in new window. The online version can be then accessed at ramanmetrix.eu/accounts/loginopen in new window with no installation required. Long-term external licenses can be requested at info@bpd.gmbh.
