# Online Version

# Access

The online version of RAMANMETRIX can be found at ramanmetrix.euopen in new window.

Please refer to ramanmetrix.eu/new_requestopen in new window for access.

# Advantages

The graphical user interface of the online version is identical to the desktop version.

The online version does not require installation and is not dependent on the operating system. It can be used on any electronic device with a modern browser and internet connection.

# Limitations of the online version

The online version requires stable internet connection.

Integration with other software is very limited. The desktop version is required for using the backend APIs and the pyRamanmetrix tools.
