# Introduction

RAMANMETRIX software performs a one-click data analysis of Raman spectroscopic data in a robust and reliable way. A convenient and simple user interface provides easy navigation through all necessary data processing steps. At each processing step, overview plots are produced to make controlling by an experienced user more convenient. These plots can be saved in PNG and CSV formats or summarized within a single report. The parameter of the processing steps can be controlled through a user-friendly graphical interface. The complexity of this interface can be adjusted accordingly to the userâ€™s expertise level. Users can upload data and metadata, standardize Raman spectral data and construct various machine learning models including deep learning for Raman spectroscopic data. Afterwards, these classification or regression models can be applied to test data sets, which can be unlabeled.
