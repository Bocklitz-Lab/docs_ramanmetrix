RAMANMETRIX

The software performs a one-click data analysis of Raman spectroscopic data in a robust and reliable way.

Documentation →

Simple

upload data and metadata; standardize Raman spectral data and construct various machine learning models including deep learning for Raman spectroscopic data; apply models on test data sets.

Modern GUI

easy navigation through all necessary data processing steps; overview plots are produced to allow a controlling by an experienced user; save plots in PNG and CSV formats or generate a single PDF report.
