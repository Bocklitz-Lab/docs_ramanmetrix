Ramanmetrix software

The software performs a one-click data analysis of Raman spectroscopic data in a robust and reliable way.

Get Started →

Simple

Users can upload data and metadata, standardize Raman spectral data and construct various machine learning models including deep learning for Raman spectroscopic data. Models can be applied on test data sets.

Modern GUI

A convenient and simple user interface provides easy navigation through all necessary data processing steps. The parameters of the processing steps can be controlled through a user-friendly graphical interface.

Saving results

At each processing step overview plots are produced to allow a controlling by an experienced user. These plots can be saved in PNG and CSV formats or summarized within a single PDF report.
